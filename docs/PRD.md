# WordCore 产品需求文档

> 版本：v0.1 · 日期：2026-03-07

---

## 一、产品概述

WordCore 是一款面向英语学习者的核心词汇练习 Web 应用。通过"看单词 → 写例句 → 对比原句"的学习闭环，帮助用户真正掌握高频英语词汇的实际用法。

---

## 二、目标用户

有一定英语基础、希望巩固和内化高频词汇的学习者。

---

## 三、核心功能

### 3.1 词库

- 收录英语使用频率排名前 **3000** 的核心词汇
- 每个词条包含：
  - 单词（word）
  - 词性（part of speech）
  - 中文释义（definition）
  - 参考例句（example sentence，简短、常见场景）

### 3.2 学习卡片

每次展示一张单词卡，内容布局：

```
[ 单词 ]
[ 词性 · 中文释义 ]
─────────────────────
[ 参考例句（默认遮挡）]
```

交互流程：

1. 用户看到单词和释义
2. 用户在输入框中**自己写一句例句**
3. 点击"对比"后，参考例句显示，用户自行判断是否表达一致
4. 用户点击 **"掌握"** 或 **"再练"** 决定该词是否标记为已掌握

### 3.3 进度追踪

- 每个单词有两种状态：`未掌握` / `已掌握`
- 主页显示整体进度：已掌握数 / 3000
- 支持按状态筛选（全部 / 未掌握 / 已掌握）

---

## 四、页面结构

| 页面 | 说明 |
|------|------|
| 首页 | 显示学习进度，进入学习入口 |
| 学习页 | 单词卡片 + 例句输入 + 对比展示 |
| 词库页 | 列表浏览所有单词及掌握状态 |

---

## 五、非功能需求

- 平台：Web（移动端适配优先）
- 数据存储：本地（localStorage），无需登录
- 无广告、无账号体系

---

## 六、暂不包含的功能（超出当前范围）

- 听力 / 发音
- 单词拼写测试
- 用户账号与云同步
- 词库自定义导入
